基础指令：
mkdir 文件夹名 创建一个文件夹;
rm -rf 文件名

touch 文件名 创建一个文件; 	

clear 清屏

cd 文件夹路径/  进入对应文件;
cd .. 回退一级文件 

vim 文件名 ;

i 进入插入模式;

esc + :wq 保存并推出;

ctrl + c + c; 强制终止程序;

ls 查看当前目录下所有文件的指令;

2.** 初始化本地仓库；
	git init
3.git add 文件名；   把某个文件放入暂存区中；
	-A           帮当前目录下的所有文件都放入到暂存区中；
4.git status 查看暂存区的状态；绿色代表工作区和暂存区代码一致； 

on branch master   在master分支上

no commits yet      未能成功的提交过


5.git diff 那里发生里改变

6.git commit -m "一定要写注释" 在暂存区里把代码放入到本地仓库中产生版本；

7.git reset --hard 版本号 版本还原；

8.git log 查看版本号信息；
  git reflog 查看简要版本信息；

9. ** git remote add origin 项目路径；
	git remote 远程仓库设置；设置成add名字叫做origin（代表项目路径）项目路径；

10.git push -u origin master; 本地仓库推送到远端仓库；
